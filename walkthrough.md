# 리턴즈(Returns) 개발 워크스루

## 🚀 프로젝트 시작 가이드
이 문서는 리턴즈 플랫폼의 개발 과정과 주요 기능 구현 방법을 설명합니다.

### 1. 디자인 시스템 (src/index.css)
리턴즈는 '신뢰'와 '신속'을 핵심 가치로 합니다.
- **Primary Blue (`--primary`)**: 신뢰를 상징하며 주요 버튼과 헤더에 사용됩니다.
- **Secondary Red (`--secondary`)**: 분실물(LOST)의 긴급함을 표시합니다.
- **Glassmorphism**: 상단 헤더와 하단 네비게이션에 `backdrop-filter`를 적용하여 현대적이고 프리미엄한 느낌을 줍니다.

### 2. 레이아웃 구조 (src/App.jsx)
현재 구현된 초기 버전은 다음과 같은 구조를 가집니다:
- **Sticky Header**: 서비스 이름과 알림/검색 버튼.
- **View Switcher**: '피드'와 '지도' 뷰를 전환할 수 있는 탭.
- **Post Feed**: 카드 형태의 실종/분실 게시물 목록.
- **Bottom Navigation**: 주요 메뉴 이동을 위한 고정 바.

### 3. 주요 기능 구현 상세

#### AI 스마트 신고 (예정)
- 사용자가 사진을 업로드하면 `Google Cloud Vision API`를 통해 사물을 분석합니다.
- 예: 강아지 사진 업로드 시 -> '반려동물', '강아지', '골든 리트리버' 태그 자동 생성.

#### 지도 피드 (예정)
- 위치 기반 데이터를 바탕으로 지도 위에 핀을 표시합니다.
- 사용자의 현재 위치를 중심으로 주변 게시물을 필터링합니다.

#### 전단지 생성 (예정)
- 게시물 상세 페이지에서 '전단지 생성' 버튼을 누르면, 기성 템플릿에 데이터가 주입된 PDF 파일을 생성합니다.

## 🛠 실행 방법
1. 종속성 설치: `npm install`
2. 개발 서버 실행: `npm run dev`
3. 브라우저에서 접속: `http://localhost:5173`
